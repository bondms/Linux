Diagnosis tools: ifconfig, route, iptables, shorewall, wireshark, ip, sudo netstat -aveep, nm-tool, rpcinfo [-p], ...
Configuration tools: ifconfig, ifup, ifdown, dhclient [-r], lshw, ...

To diagnose issues it may be necessary to install tools from the distribution CD-ROM rather than the Internet repository, e.g.
if
# sudo apt-get install traceroute
fails, then try
# mount /media/cdrom
# sudo dpkg -i /media/cdrom/pool/main/t/traceroute/traceroute_2.0.12-2_i386.deb
